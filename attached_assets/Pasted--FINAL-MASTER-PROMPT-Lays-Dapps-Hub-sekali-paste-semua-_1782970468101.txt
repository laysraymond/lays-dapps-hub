## Part 1 — Builder Leaderboard (`/builders`)

Add a new page `/builders` showing a leaderboard of dApp builders ranked by combined impact.

**Add "Builders" to the existing navbar** between "Leaderboard" and "Submit", same styling as existing nav items.

**New API route `/api/builders`:**
Query the `Dapp` table, aggregate by `creatorHandle`, return top 100:

```ts
SELECT 
  creatorHandle,
  COUNT(*) as totalDapps,
  SUM(clickCount) as totalClicks,
  MAX(createdAt) as latestDapp
FROM Dapp
WHERE approved = true
GROUP BY creatorHandle
ORDER BY totalClicks DESC, totalDapps DESC
LIMIT 100
```

Return JSON array, each item:
```ts
{
  handle: string,       // e.g. "kency17"
  totalDapps: number,
  totalClicks: number,
  latestDapp: string,   // name of most recently added dapp
  xUrl: string,         // "https://x.com/" + handle (strip any leading @)
}
```
